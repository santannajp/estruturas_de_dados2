# estruturas_de_dados2

Estudo de estrutura de dados atraves da construção de um pipeline para transformar dados de json para parquet/csv.

# Tecnologias

- Python 3.11

# Estrutura do projeto

- etl.py: contém as funções de extração, transformação e carregamento
- pipeline.py: contém o pipeline que orquestra as funções de etl.py
- data: contém os arquivos json que serão processados
